# Python Powered Dashboard for Easy Data Visualization

This is a simple dashboard that utilizes modules in the `modules` folder to display various data in a user-friendly manner. For example, the weather module uses the OpenWeatherMap API to showcase the current weather in a specified location.

## Installation

To install the necessary dependencies, run the following command:

```bash
pip install -r requirements.txt
```

## Usage

To start the dashboard, execute the following command:

```bash
python app.py
```
